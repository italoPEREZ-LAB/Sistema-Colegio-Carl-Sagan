# Sistema Colegio Carl Sagan

## Descripción

Sistema web para la gestión académica del Colegio Carl Sagan.

## Tecnologías Utilizadas

### Backend

* Java 17
* Spring Boot
* Spring Security
* Spring Data JPA
* MySQL

### Frontend

* Angular
* TypeScript
* Bootstrap

## Funcionalidades

* Gestión de Alumnos
* Gestión de Tutores
* Gestión de Grados
* Gestión de Usuarios
* Inicio de Sesión

## Base de Datos

MySQL

Base de datos utilizada:

```sql
Carl_Sagan
```

## Ejecución del Backend

1. Crear la base de datos `Carl_Sagan`.
2. Configurar `application.properties`.
3. Ejecutar `ColegioApplication.java`.

## Ejecución del Frontend

```bash
npm install
npm start
```

Abrir:

```text
http://localhost:4200
```
